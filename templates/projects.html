Calculator

Basic arithmetic operations

Beginner

Build a command-line calculator that performs basic arithmetic operations. Perfect for learning input handling, functions, and error management.

  • Addition, subtraction, multiplication, division
  • Input validation and error handling
  • Modular function design
  • User-friendly interface
Functions Input/Output Error Handling

Problem Statement

Create a calculator program that can perform basic arithmetic operations (addition, subtraction, multiplication, division) with proper error handling for invalid inputs and division by zero.

Solution

def add(x, y):
    """Add two numbers"""
    return x + y

def subtract(x, y):
    """Subtract y from x"""
    return x - y

def multiply(x, y):
    """Multiply two numbers"""
    return x * y

def divide(x, y):
    """Divide x by y with error handling"""
    if y == 0:
        return "Error: Cannot divide by zero"
    return x / y

def main():
    print("Python Calculator")
    print("================")
    
    while True:
        print("\nOperations:")
        print("1. Addition (+)")
        print("2. Subtraction (-)")
        print("3. Multiplication (*)")
        print("4. Division (/)")
        print("5. Exit")
        
        choice = input("Enter choice (1-5): ")
        
        if choice == '5':
            print("Thank you for using the calculator!")
            break
        
        if choice not in ['1', '2', '3', '4']:
            print("Invalid choice! Please try again.")
            continue
        
        try:
            num1 = float(input("Enter first number: "))
            num2 = float(input("Enter second number: "))
        except ValueError:
            print("Invalid input! Please enter numbers only.")
            continue
        
        if choice == '1':
            result = add(num1, num2)
            operation = '+'
        elif choice == '2':
            result = subtract(num1, num2)
            operation = '-'
        elif choice == '3':
            result = multiply(num1, num2)
            operation = '*'
        elif choice == '4':
            result = divide(num1, num2)
            operation = '/'
        
        if isinstance(result, str):  # Error message
            print(result)
        else:
            print(f"Result: {num1} {operation} {num2} = {result}")

if __name__ == "__main__":
    main()

Learning Outcomes

  • Function definition and parameter passing
  • User input handling and validation
  • Error handling with try-except blocks
  • Control flow with while loops and conditionals
  • Modular programming practices

Number Guessing Game

Interactive guessing game

Beginner

Create an interactive number guessing game where the computer randomly selects a number and the player has to guess it with feedback and limited attempts.

  • Random number generation
  • Multiple difficulty levels
  • Attempt tracking and scoring
  • Colorful feedback messages
Random Module While Loops Game Logic

Problem Statement

Create a number guessing game where the computer generates a random number between 1 and 100. The player gets feedback (too high/too low) and has a limited number of attempts to guess correctly.

Solution

import random

def guess_the_number():
    """Main game function"""
    print("üéØ Welcome to Number Guessing Game!")
    print("====================================")
    
    # Game settings
    min_number = 1
    max_number = 100
    max_attempts = 10
    
    # Generate random number
    secret_number = random.randint(min_number, max_number)
    
    print(f"I'm thinking of a number between {min_number} and {max_number}")
    print(f"You have {max_attempts} attempts to guess it!")
    
    attempts = 0
    guessed = False
    
    while attempts < max_attempts and not guessed:
        try:
            guess = int(input(f"\nAttempt {attempts + 1}/{max_attempts}: Enter your guess: "))
            
            if guess < min_number or guess > max_number:
                print(f"‚ö†Ô∏è  Please enter a number between {min_number} and {max_number}!")
                continue
                
            attempts += 1
            
            if guess == secret_number:
                print(f"üéâ Congratulations! You guessed it in {attempts} attempts!")
                guessed = True
            elif guess < secret_number:
                print("üìà Too low! Try a higher number.")
                remaining = max_attempts - attempts
                print(f"Attempts remaining: {remaining}")
            else:
                print("üìâ Too high! Try a lower number.")
                remaining = max_attempts - attempts
                print(f"Attempts remaining: {remaining}")
                
        except ValueError:
            print("‚ùå Invalid input! Please enter a valid number.")
    
    if not guessed:
        print(f"\nüòî Game Over! The number was {secret_number}")
        print("Better luck next time!")

def main():
    play_again = 'y'
    
    while play_again.lower() == 'y':
        guess_the_number()
        
        print("\n" + "="*40)
        play_again = input("Would you like to play again? (y/n): ")
        
        while play_again.lower() not in ['y', 'n']:
            play_again = input("Please enter 'y' for yes or 'n' for no: ")
    
    print("\nüëã Thanks for playing! See you next time!")

if __name__ == "__main__":
    main()

Learning Outcomes

  • Random number generation with random module
  • While loop implementation for game loops
  • Exception handling for invalid inputs
  • Conditional logic for game feedback
  • User experience design for interactive programs

Password Generator

Secure password creation tool

Create a secure password generator that creates strong passwords with customizable length and character sets including letters, numbers, and special characters.

  • Customizable password length
  • Character set selection
  • Password strength validation
  • Multiple password generation
String Module Random Generation Security

Problem Statement

Create a password generator that can generate secure passwords with customizable options including length, inclusion of uppercase/lowercase letters, numbers, and special characters.

Solution

import random
import string

def generate_password(length=12, use_uppercase=True, use_lowercase=True, 
                     use_numbers=True, use_symbols=True):
    """
    Generate a random password with specified criteria
    
    Args:
        length (int): Length of the password
        use_uppercase (bool): Include uppercase letters
        use_lowercase (bool): Include lowercase letters  
        use_numbers (bool): Include numbers
        use_symbols (bool): Include special symbols
    
    Returns:
        str: Generated password
    """
    if length < 4:
        raise ValueError("Password length must be at least 4 characters")
    
    # Build character set
    characters = ""
    
    if use_uppercase:
        characters += string.ascii_uppercase
    if use_lowercase:
        characters += string.ascii_lowercase
    if use_numbers:
        characters += string.digits
    if use_symbols:
        characters += "!@#$%^&*()_+-=[]{}|;:,.<>?"
    
    if not characters:
        raise ValueError("At least one character type must be selected")
    
    # Generate password
    password = ''.join(random.choice(characters) for _ in range(length))
    
    return password

def check_password_strength(password):
    """
    Check the strength of a password
    
    Args:
        password (str): Password to check
    
    Returns:
        dict: Password strength analysis
    """
    strength = {
        'length': len(password) >= 8,
        'uppercase': any(c.isupper() for c in password),
        'lowercase': any(c.islower() for c in password),
        'numbers': any(c.isdigit() for c in password),
        'symbols': any(c in "!@#$%^&*()_+-=[]{}|;:,.<>?" for c in password)
    }
    
    score = sum(strength.values())
    
    if score >= 5:
        strength_level = "Very Strong"
    elif score >= 4:
        strength_level = "Strong"
    elif score >= 3:
        strength_level = "Medium"
    else:
        strength_level = "Weak"
    
    return {
        'strength_level': strength_level,
        'score': score,
        'criteria': strength
    }

def main():
    print("üîê Password Generator")
    print("====================")
    
    try:
        # Get user preferences
        length = int(input("Enter password length (default 12): ") or 12)
        
        print("\nCharacter sets:")
        use_upper = input("Include uppercase letters? (y/n, default y): ").lower() != 'n'
        use_lower = input("Include lowercase letters? (y/n, default y): ").lower() != 'n'
        use_numbers = input("Include numbers? (y/n, default y): ").lower() != 'n'
        use_symbols = input("Include special symbols? (y/n, default y): ").lower() != 'n'
        
        # Generate password
        password = generate_password(
            length=length,
            use_uppercase=use_upper,
            use_lowercase=use_lower,
            use_numbers=use_numbers,
            use_symbols=use_symbols
        )
        
        print(f"\nüéâ Generated Password: {password}")
        
        # Check strength
        strength_analysis = check_password_strength(password)
        print(f"\nüìä Password Strength: {strength_analysis['strength_level']}")
        print(f"Score: {strength_analysis['score']}/5")
        
        print("\nCriteria met:")
        criteria = strength_analysis['criteria']
        print(f"  ‚úì Length (8+ chars): {'Yes' if criteria['length'] else 'No'}")
        print(f"  ‚úì Uppercase letters: {'Yes' if criteria['uppercase'] else 'No'}")
        print(f"  ‚úì Lowercase letters: {'Yes' if criteria['lowercase'] else 'No'}")
        print(f"  ‚úì Numbers: {'Yes' if criteria['numbers'] else 'No'}")
        print(f"  ‚úì Special symbols: {'Yes' if criteria['symbols'] else 'No'}")
        
        # Option to generate multiple passwords
        generate_more = input("\nGenerate another password? (y/n): ").lower() == 'y'
        
        if generate_more:
            num_passwords = int(input("How many passwords to generate? (1-10): ") or 3)
            
            print(f"\nüîÑ Generating {num_passwords} passwords:")
            for i in range(num_passwords):
                new_password = generate_password(
                    length=length,
                    use_uppercase=use_upper,
                    use_lowercase=use_lower,
                    use_numbers=use_numbers,
                    use_symbols=use_symbols
                )
                print(f"  {i+1}. {new_password}")
    
    except ValueError as e:
        print(f"‚ùå Error: {e}")
    except KeyboardInterrupt:
        print("\n\nüëã Program interrupted by user.")

if __name__ == "__main__":
    main()

Learning Outcomes

  • String manipulation and character sets
  • Function parameterization and validation
  • Random generation techniques
  • Password security best practices
  • User input validation and error handling
